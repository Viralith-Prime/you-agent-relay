🔖 Universal Bookmarklet Setup

Desktop Browser Setup

  1. Show your bookmarks bar:
    Chrome/Edge: Press Ctrl+Shift+B (Windows) or Cmd+Shift+B (Mac)
    Firefox: Right-click toolbar → "Bookmarks Toolbar" → "Always Show"
    Safari: View menu → Show Bookmarks Bar
  2. Drag this button to your bookmarks bar:
    🛡️ Guardian Inject

    Can't drag? Right-click and add to bookmarks manually

  3. Test it out:
    1. Save a prompt in the AI Hub
    2. Visit any AI site (ChatGPT, Claude, etc.)
    3. Click the "Guardian Inject" bookmark
    4. Your prompt appears instantly!

  1. Copy the bookmarklet code:
    javascript:(function(){if(window.GuardianOrchestrator){window.GuardianOrchestrator.injectPrompt(localStorage.getItem('guardian-prompt')||'',window.location.hostname);return;}const s=document.createElement('script');s.src='${window.location.origin}/guardian-orchestrator.js?t='+Date.now();s.onload=function(){const p=localStorage.getItem('guardian-prompt');if(p&&window.GuardianOrchestrator){setTimeout(()=>{window.GuardianOrchestrator.injectPrompt(p,window.location.hostname);},1000);}};document.head.appendChild(s);const i=document.createElement('div');i.style.cssText='position:fixed;top:10px;right:10px;background:#667eea;color:white;padding:10px 20px;border-radius:20px;z-index:10000;animation:pulse 2s infinite;';i.textContent='🛡️ Guardian Active';document.body.appendChild(i);const st=document.createElement('style');st.textContent='@keyframes pulse{0%,100%{opacity:1;}50%{opacity:0.7;}}';document.head.appendChild(st);setTimeout(()=>i.remove(),3000);})();

Installation Steps

  1. Download Extension Files:
    📥 Download Guardian Extension
  2. Extract the ZIP file:
    Extract to a folder you won't delete (e.g., Documents/GuardianExtension)
  3. Enable Developer Mode:
    Chrome: Go to chrome://extensions/
    Edge: Go to edge://extensions/
    Toggle "Developer mode" switch (top right)
  4. Load the extension:
    Click "Load unpacked" → Select the extracted folder → Select the "extension" subfolder
  5. Pin the extension:
    Click the puzzle piece icon → Pin Guardian Extension

⚡ Progressive Web App Setup

Install Guardian Hub as a native app on your device for the best experience!

Desktop Installation

  1. Look for the install icon:
    Chrome/Edge: Install icon in the address bar (right side)
    Firefox: Not supported yet - use bookmarklet instead
  2. Click Install:
    A popup will appear → Click "Install"
  3. Launch from:
    Windows: Start Menu or Desktop
    Mac: Applications folder or Dock
    Linux: Application menu

Mobile Installation

  1. Open Guardian Hub in Safari (must be Safari)
  2. Tap the Share button ⬆️
  3. Scroll down and tap "Add to Home Screen"
  4. Name it "Guardian Hub" and tap "Add"
  1. Open Guardian Hub in Chrome
  2. Tap the ⋮ menu
  3. Tap "Add to Home screen" or "Install app"
  4. Confirm installation
